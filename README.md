# AviMapsLite

A React Native app displaying [Open Flight Map for Finland](https://www.openflightmaps.org/ef-finland/?airac=2010&language=local)

### Features

- Offline maps
- Lock to plane GPS location
- Free rotation of map
- Heading, altitude, ETA display
